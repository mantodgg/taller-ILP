# Taller: Programación Lineal Entera (ILP) en Python

Repositorio desarrollado en el marco del Club de Datos (clase del 14/4/2026).

## ¿De qué trata?

Se exploran problemas de optimización resueltos con **Programación Lineal Entera (ILP)**,
utilizando la librería `scipy.optimize` de Python (función `milp`).

## Problemas resueltos

1. **Imprenta** — maximizar la ganancia en la producción de folletos y afiches sujeto a 
restricciones de cantidad, papel y costo.
2. **Fábrica de cajas** — maximizar la ganancia en la producción de cajas chicas, medianas 
y grandes con recursos limitados de cartón y cinta.
3. **Reparto de alumnos por letra** — asignar grupos de estudiantes a docentes respetando 
que las iniciales no se separen y que los grupos queden balanceados (problema de asignación).
4. **Reparto de puestos de trabajo** — asignar personas a empleos maximizando la suma de 
puntajes de aptitud (problema de asignación óptima).
